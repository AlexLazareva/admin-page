### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
Необходимо создать компонент админки для отображения, редактирования и сохранения полей пользователя. 
Страница генерируется на основе объекта, в котором ханится информация о всех полях.
В зависимости от типа поля меняется отображение: если string, то отображается текстовое поле ввода, если date - дата и возможность выбрать из календаря.
